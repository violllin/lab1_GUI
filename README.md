### Определение КС-граммитики:
````
1. E → TA
2. A → ε | + TA | - TA
3. T → FB
4. B → ε | * FB | / FB | % FB
5. F → num | id | (E)
6. id → letter {letter | digit | _ }
7. num → digit {digit}
````
### Примеры корректных строк: 
(a + b) * c

10 / (5 - 3)

((x + 1) * (y - 2)) % 10
