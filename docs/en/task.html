Problem Statement

Option 87: Lambda expression in the Swift language.

The goal of this work is the design and programmatic implementation of a syntax analyzer (parser) to verify the correctness of a code fragment representing a Swift lambda expression.

Main Requirements:

  • Development of a context-free grammar for the Lambda expression.
  • Implementation of the recursive descent method for analyzing the input string of symbols.
  • Providing diagnostics and neutralization of syntax errors using the Irons method.

Expression Format

In accordance with the grammar, a correct expression must have the following form:

let variable_name = { (parameter: type) -> return_type in return expression };

Examples of correct strings for verification:

  • let calc = { (a: Int, b: Int) -> Int in return a + b };
  • let simple = { (s: String) -> String in return s };
