Grammar Classification

According to Chomsky's classification, the resulting generative grammar G[<START>] corresponds to the type of context-free grammars.

This is justified by the fact that all production (reduction) rules correspond to the following form:

A → α

where:

  • A — strictly one non-terminal symbol from the vocabulary VN;
  • α — any string consisting of terminal and non-terminal symbols of the combined vocabulary.

Additional characteristics:

The grammar is also unambiguous and deterministic, which allows it to be implemented using the recursive descent method without the need for backtracking.
