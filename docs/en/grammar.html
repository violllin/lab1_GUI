Grammar Development

The grammar for a Swift language lambda expression G[<START>] in Chomsky notation with productions P:

1.  <START>        -> ‘LET’ <LET>
2.  <LET>          -> ‘ ’ <SPACE_1>
3.  <SPACE_1>      -> <IDENTIFIER> <VAR_NAME>
4.  <VAR_NAME>     -> ‘=’ <ASSIGN>
5.  <ASSIGN>       -> ‘{’ <LBRACE>
6.  <LBRACE>       -> ‘(’ <LPAREN>
7.  <LPAREN>       -> <PARAM> <PARAM_LIST>
8.  <PARAM>        -> <IDENTIFIER> ‘:’ <TYPE>
9.  <TYPE>         -> ‘Int’ | ‘String’ | ‘Float’ | ‘Bool’
10. <PARAM_LIST>   -> ‘,’ <LPAREN> | ‘)’ <RPAREN>
11. <RPAREN>       -> ‘->’ <ARROW>
12. <ARROW>        -> <TYPE> <RETURN_TYPE>
13. <RETURN_TYPE>  -> ‘ ’ <SPACE_2>
14. <SPACE_2>      -> ‘in’ <IN>
15. <IN>           -> ‘ ’ <SPACE_3>
16. <SPACE_3>      -> ‘return’ <RETURN>
17. <RETURN>       -> ‘ ’ <SPACE_4>
18. <SPACE_4>      -> <EXPR> <CLOSE>
19. <EXPR>         -> <TERM> <EXPR_TAIL>
20. <EXPR_TAIL>    -> ‘+’ <TERM> <EXPR_TAIL> | ‘-’ <TERM> <EXPR_TAIL> | ε
21. <TERM>         -> <FACTOR> <TERM_TAIL>
22. <TERM_TAIL>    -> ‘*’ <FACTOR> <TERM_TAIL> | ‘/’ <FACTOR> <TERM_TAIL> | ε
23. <FACTOR>       -> <IDENTIFIER> | <NUMBER> | (<EXPR>)
24. <CLOSE>        -> ‘}’ <END>
25. <END>          -> ‘;’
26. <IDENTIFIER>   -> letter <ID_TAIL>
27. <ID_TAIL>      -> letter <ID_TAIL> | digit <ID_TAIL> | ε
28. <NUMBER>       -> digit <NUM_TAIL>
29. <NUM_TAIL>     -> digit <NUM_TAIL> | ε

Grammar Components:

  • V_T (Terminal symbols): {'let', '=', '{', '(', ')', ':', ',', '->', 'in', 'return', '+', '-', '*', '/', ';', 'Int', 'String', 'Float', 'Bool', letter, digit}.
  • V_N (Non-terminal symbols): {START, LET, SPACE_1, VAR_NAME, ASSIGN, LBRACE, LPAREN, PARAM, TYPE, PARAM_LIST, RPAREN, ARROW, RETURN_TYPE, SPACE_2, IN, SPACE_3, RETURN, SPACE_4, EXPR, EXPR_TAIL, TERM, TERM_TAIL, FACTOR, CLOSE, END, IDENTIFIER, ID_TAIL, NUMBER, NUM_TAIL}.
  • Z (Target symbol): <START>.
