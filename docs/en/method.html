Analysis Method

The recursive descent method was chosen to implement the syntax analyzer. This method belongs to the class of top-down parsing algorithms.

Core Principles of the Method:

  • Each non-terminal symbol of the grammar G[<START>] is associated with a separate program function.
  • The analysis process begins by calling the function corresponding to the target symbol of the grammar.
  • The parse tree is constructed from the root (target symbol) to the leaves (terminal symbols).
  • The analyzer reads the current token and, based on it, decides which branch of the rule to choose.

Operational Algorithm:

  • When a non-terminal symbol is encountered in the right side of a rule, a recursive call to its corresponding function occurs.
  • When a terminal symbol is encountered, the analyzer compares it with the current symbol of the input string.
  • If the symbols match, the current position pointer moves to the next token.
  • In case of a mismatch, a procedure for processing and neutralizing syntax errors is initiated.
