Вариант 87: Лямбда-выражение на языке Swift.

Целью работы является проектирование и программная реализация синтаксического анализатора (парсера) для проверки корректности фрагмента кода, представляющего собой лямбда-выражение Swift.

Основные требования:

  • Разработка контекстно-свободной грамматики для Лямбда-варажение.
  • Реализация метода рекурсивного спуска для анализа входной цепочки символов.
  • Обеспечение диагностики и нейтрализации синтаксических ошибок методом Айронса.

Формат выражения

В соответствии с грамматикой, корректное выражение должно иметь вид:

let имя_переменной = { (параметр: тип) -> тип_возврата in return выражение };

Примеры корректных строк для проверки:

  • let calc = { (a: Int, b: Int) -> Int in return a + b };
  • let simple = { (s: String) -> String in return s };
